# Telegram Бот-Мотиватор

## Описание
Бот для Telegram, который предоставляет мотивирующие фразы и умные цитаты. Поддерживает быстрые кнопки для получения новой цитаты или мотивации. При старте бот отправляет приветственное сообщение с картинкой. Пользователи могут добавлять свои собственные фразы и цитаты, а также получать цитаты из внешнего API.

##Функциональность
- Получение мотивирующих фраз и цитат
- Добавление пользовательских фраз и цитат
- Получение цитат из внешнего источника (API)
- Уникальные очереди цитат и фраз для каждого пользователя
- Интерактивные кнопки для быстрого доступа к контенту

## Запуск
1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Создайте файл `.env` в корне проекта и добавьте в него строку:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   ```
3. Убедитесь, что в папке проекта есть файл `1754391065.png` (он будет отправляться при /start).
4. Запустите бота:
   ```
   python main.py
   ```

## Команды
- `/start` — приветственное сообщение с картинкой и инструкция
- `/getmotivation` — получить мотивирующую фразу (с кнопками)
- `/getsmartquote` — получить мотивирующую цитату (с кнопками)
- `/externalquote` — получить цитату из внешнего API (с кнопками)
- `/addmotivation` <текст> — добавить свою мотивирующую фразу
- `/addquote` <текст> — добавить свою цитату

## Быстрые кнопки
Под каждым сообщением (кроме приветствия) отображаются две кнопки:
- **Цитата дня** — получить случайную цитату
- **Мой мотиватор** — получить мотивирующую фразу
